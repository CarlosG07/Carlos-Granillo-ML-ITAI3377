### IIoT Time Series Forecasting (Lab Project)
- Overview

This project focuses on applying time-series forecasting techniques to Industrial Internet of Things (IIoT) temperature data. It demonstrates how data preprocessing, feature engineering, and automated machine learning can be combined to generate accurate predictions for real-world IoT systems.

- Objectives
- Forecast IIoT sensor data (e.g., temperature)
- Improve prediction accuracy using preprocessing and feature engineering
- Evaluate model performance using automated tools
- Explore generative AI techniques for data enhancement

- Key Concepts
- Time-series forecasting
- Data preprocessing (cleaning, missing values, timestamps)
- Feature engineering (time-based features, rolling averages)
- Automated ML (Nixtla AutoML / TimeGPT)
- Generative AI (Variational Autoencoder - VAE)
- Methodology

- 1. Data Preparation
- Cleaned dataset
- Handled missing values
- Verified timestamps
- Removed inconsistencies

- 2. Feature Engineering
- Time-based features (hour, trends)
- Rolling averages
- Enhanced input representation

- 3. Model Training
- Used TimeGPT (Nixtla AutoML)
- Automated model selection and tuning
- Compared forecasting performance

- 4. Generative AI Enhancement
- Applied Variational Autoencoder (VAE)
- Generated synthetic data
- Improved training dataset size and quality

- Evaluation Metrics
- MAE (Mean Absolute Error)
- RMSE (Root Mean Square Error)

- Key Insights
- Data quality heavily impacts forecasting accuracy
- Feature engineering significantly improves results
- Automated ML simplifies model selection
- Synthetic data can enhance performance

- Results
- Improved prediction accuracy after preprocessing
- Better trend detection using engineered features
- Enhanced dataset using generative models

- Future Improvements
- Incorporate additional sensor data
- Use advanced deep learning models
Implement real-time forecasting pipelines
