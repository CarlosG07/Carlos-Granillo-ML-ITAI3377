# Lab 2 – Neural Network with MNIST

## Objective
The objective of this lab was to build, train, and deploy a neural network model using the MNIST dataset. This lab focused on understanding the full machine learning pipeline, from data preparation to deployment.

## What I Did
- Loaded the MNIST dataset of handwritten digits
- Normalized pixel values between 0 and 1
- Built a neural network using TensorFlow and Keras
- Trained the model and evaluated its accuracy
- Saved the trained model
- Converted the model into TensorFlow Lite format
- Ran predictions using a TensorFlow Lite interpreter

## What I Learned
This lab helped me understand how neural networks recognize patterns in image data. I also learned how important preprocessing is when working with machine learning models. Converting the model to TensorFlow Lite showed how AI can run on edge devices like smartphones and IoT systems.

## Challenges
One of the main challenges was handling data shapes and formatting. Small mistakes in tensor dimensions caused errors when running the model. Another challenge was understanding the difference between TensorFlow and TensorFlow Lite and how each is used.
