# Assignment 1 – Development Tools for Edge AI and IoT

## Objective
The goal of this assignment was to explore key development tools used in Edge AI and IoT systems. These tools are essential for building systems that process data locally instead of relying entirely on cloud computing.

## Topics Covered
This assignment focused on the following tools:
- Visual Studio Code (VS Code)
- Node.js
- Edge Impulse CLI
- TensorFlow and TensorFlow Lite
- Google Colab
- Generative AI coding tools

## What I Learned
I learned how different tools work together in developing Edge AI systems. For example:
- VS Code is used for writing and debugging code
- Node.js helps manage real-time communication between devices
- TensorFlow is used to train machine learning models
- TensorFlow Lite is used to deploy models on edge devices
- Google Colab allows cloud-based experimentation

## Challenges
Understanding how all these tools connect in a real system was challenging. It required thinking about both software development and hardware limitations in IoT environments.

## Real-World Application
These tools are commonly used in systems like:
- Smart homes
- Wearable devices
- Industrial automation systems
- Security cameras with on-device AI
