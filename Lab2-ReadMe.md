# Lab 2 – Neural Network with MNIST

## Overview
In this lab, I built, trained, and deployed a simple neural network using the MNIST handwritten digit dataset. The goal was to understand the full machine learning process from data preparation to deployment.

## What I Did
- Loaded and normalized the MNIST dataset
- Built a neural network using TensorFlow and Keras
- Trained the model and evaluated accuracy
- Converted the model to TensorFlow Lite format
- Ran predictions using a TensorFlow Lite interpreter

## What I Learned
I learned how neural networks recognize patterns in image data and how important data preprocessing is. I also learned how to convert models into TensorFlow Lite so they can run efficiently on edge devices like smartphones or IoT systems.

## Challenges
One challenge was handling data shapes and formatting. Small mistakes in tensor shapes or data types caused errors when using the TensorFlow Lite interpreter. Understanding the difference between TensorFlow and TensorFlow Lite was also difficult at first.
